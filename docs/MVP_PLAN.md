# MVP Plan

## Scope

The first playable version should prove the tone and interaction loop, not the full museum.

## Required Rooms

- Grand Lobby
- Hall of Absurd Liability
- Bureaucracy Wing
- Pun Sanitation Chamber

## Required Interactables

- Passport kiosk
- Liability waiver kiosk
- Bureaucracy clerk
- Pun plaque
- Sanitation booth

## Prototype Success Criteria

- Player understands the premise within 60 seconds.
- At least three interactions create funny consequences.
- Player can complete one absurd loop from passport to stamp.
- The implementation is simple enough to debug inside Roblox Studio.

## First Milestone

```text
Spawn -> Get Passport -> Sign Waiver -> Get Status -> Talk to Clerk -> Trigger Pun -> Sanitize -> Earn Stamp
```
