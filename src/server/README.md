# Server Services

Server-side services live here and are synced into `ServerScriptService`.

The current prototype still uses `PrototypeInteractionService.server.lua` as a simple first playable loop. As the project grows, split behavior into focused services:

- `DataService`
- `PassportService`
- `QuestService`
- `DialogueService`
- `StatusEffectService`
- `ChinchillaService`
